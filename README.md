# Web Development SWE

This organization contains repositories related to the **Software Engineering Web Development** course, including labs, assignments, and projects completed throughout the course.

## Repository Structure

### Basic HTML
Introduction to web development concepts using HTML. Covers page structure, semantic elements, forms, tables, and basic web page creation.

### Web App Lab 1
A foundational web application project demonstrating front-end development concepts and interactive user interfaces.

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

The repositories in this organization are designed to:

- Practice fundamental web development skills
- Complete course lab exercises and assignments
- Build practical web application projects
- Learn modern front-end development techniques

## Course Topics

- HTML Fundamentals
- CSS Styling and Layout
- JavaScript Programming
- Responsive Web Design
- Web Application Development
- Front-End Best Practices
